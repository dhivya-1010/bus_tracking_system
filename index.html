About The Project

This system uses ESP32 integrated with RFID technology to automatically detect student bus boarding and deboarding. Each RFID scan sends real-time data to the server, triggers Gmail email notifications to parents, and updates GPS location tracking. The web dashboard enables administrators to monitor live student attendance and bus movement efficiently.
